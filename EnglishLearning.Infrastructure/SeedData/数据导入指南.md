# 人教版小学英语题库数据导入指南

## 概述

本指南介绍如何从官方渠道获取人教版 (PEP) 小学英语教材资源，并导入到系统中生成完整的题库数据。

---

## 官方数据获取渠道

### 1. 人民教育出版社官网
- **网址**: https://www.pep.com.cn/
- **资源**: 电子课本、教师用书、配套练习
- **获取方式**: 
  1. 访问人教社官网
  2. 进入"产品中心" → "数字教材"
  3. 选择小学英语 (PEP) 对应年级
  4. 下载电子课本或在线阅读

### 2. 人教社数字教材中心
- **网址**: https://jczx.pep.com.cn/
- **资源**: 正版数字教材、教学资源包
- **获取方式**: 注册账号后可下载官方数字资源

### 3. 国家中小学智慧教育平台
- **网址**: https://www.zxx.edu.cn/
- **资源**: 课程教材、教学资源
- **获取方式**: 免费注册后可访问教材资源

## 数据导入流程

```
1. 访问官网 → 2. 复制单词表 → 3. 填入 CSV 模板 → 4. 运行导入脚本
```

### 步骤 1：从官网复制单词表

访问上述任意官方资源网站，找到对应年级和单元的单词表。

例如：三年级上册 Unit 1 Hello!
```
ruler     /ˈruːlə/     尺子
pencil    /ˈpensl/     铅笔
eraser    /ɪˈreɪzə/    橡皮
...
```

### 步骤 2：使用 CSV 模板整理数据

打开 `words-template.csv` 文件，按格式填入数据：

```csv
grade,semester,unit_no,unit_name,word,phonetic_uk,phonetic_us,meaning_cn,part_of_speech,example_en,example_cn
3,上，1,Unit 1 Hello!,ruler,/ˈruːlə/,/ˈruːlər/,尺子，n.,I have a ruler.,我有一把尺子。
3,上，1,Unit 1 Hello!,pencil,/ˈpensl/,/ˈpensl/,铅笔，n.,This is my pencil.,这是我的铅笔。
```

**字段说明**：
| 字段 | 说明 | 示例 |
|------|------|------|
| grade | 年级 | 3/4/5/6 |
| semester | 学期 | 上/下 |
| unit_no | 单元号 | 1-6 |
| unit_name | 单元名称 | Unit 1 Hello! |
| word | 单词 | ruler |
| phonetic_uk | 英式音标 | /ˈruːlə/ |
| phonetic_us | 美式音标 | /ˈruːlər/ |
| meaning_cn | 中文意思 | 尺子 |
| part_of_speech | 词性 | n. |
| example_en | 英文例句 | I have a ruler. |
| example_cn | 中文例句 | 我有一把尺子。 |

### 步骤 3：运行导入脚本

```bash
# 安装依赖（如果还没安装）
pip install pyodbc pandas openpyxl

# 导入 CSV 文件
python import_words.py words-data.csv

# 或导入 Excel 文件
python import_words.py words-data.xlsx

# 或导入 JSON 文件
python import_words.py textbook-words.json
```

### 步骤 4：验证导入结果

```bash
# 查看题库统计
curl http://localhost:5000/api/seed/stats
```

返回示例：
```json
{
  "success": true,
  "data": {
    "gradeUnits": 48,
    "words": 450,
    "grammars": 50,
    "questions": 200
  }
}
```

## 支持的文件格式

### CSV 格式（推荐）
```csv
grade,semester,unit_no,unit_name,word,phonetic_uk,phonetic_us,meaning_cn,part_of_speech,example_en,example_cn
3,上，1,Unit 1 Hello!,ruler,/ˈruːlə/,/ˈruːlər/,尺子，n.,I have a ruler.,我有一把尺子。
```

### Excel 格式
与 CSV 相同的列结构，使用 `.xlsx` 或 `.xls` 扩展名。

### JSON 格式
```json
{
  "data": {
    "gradeUnits": [
      {
        "grade": 3,
        "semester": "上",
        "unitNo": 1,
        "unitName": "Unit 1 Hello!",
        "words": [
          {
            "word": "ruler",
            "phonetic_uk": "/ˈruːlə/",
            "phonetic_us": "/ˈruːlər/",
            "meaning_cn": "尺子",
            "part_of_speech": "n.",
            "example_en": "I have a ruler.",
            "example_cn": "我有一把尺子。"
          }
        ]
      }
    ]
  }
}
```

## 注意事项

1. **数据库准备**: 确保 `tb_grade_unit` 表中已有年级单元数据
2. **数据校对**: 导入前仔细核对单词、音标、释义的准确性
3. **备份数据**: 导入前备份数据库，防止数据丢失
4. **分批导入**: 建议按年级或学期分批导入，便于校验
5. **版权说明**: 教材内容版权归人民教育出版社所有，请仅用于个人学习或教学用途

## 故障排查

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| 导入脚本报错 "模块不存在" | 未安装 Python 依赖 | 运行 `pip install pyodbc pandas openpyxl` |
| 数据库连接失败 | SQL Server 未启动 | 检查 SQL Server 服务状态 |
| 中文乱码 | CSV 编码不正确 | 确保使用 UTF-8 编码保存 CSV |
| 单元不存在 | 数据库缺少年级单元数据 | 先运行年级单元导入脚本 |
| 导入后单词数为 0 | 文件路径错误或格式不对 | 检查文件路径和格式 |

---

## 技术支持

如有问题，请检查：
1. 数据库连接是否正常
2. 文件格式是否正确
3. 年级单元数据是否已存在于数据库中
4. 查看日志输出定位错误
