## 数据来源说明

由于网络限制无法直接从以下来源获取数据：
- GitHub: TapXWorld/ChinaTextbook
- 人教社官网：https://www.pep.com.cn/
- 国家中小学智慧教育平台：https://www.zxx.edu.cn/

当前数据是**根据人教版 PEP 教材大纲整理的标准单词表**，包含：
- 单词拼写
- 英式/美式音标
- 中文释义
- 词性
- 英文例句
- 中文例句

## 使用方法

### 查看题库统计
```bash
curl http://localhost:5000/api/seed/stats
```

### 重新导入某个年级
```bash
# 三年级
python import-grade3-words.py

# 四年级
python import-grade4-words.py

# 五年级
python import-grade5-words.py

# 六年级
python import-grade6-words.py
```

### 使用 CSV 导入自定义数据
```bash
python import_words.py your-data.csv
```
