# 人教版小学英语题库数据导入模板

## 快速开始

### 步骤 1：从官方资源获取单词表

访问以下任一官方网站获取教材资源：

1. **人教社官网**: https://www.pep.com.cn/
   - 进入"产品中心" → "数字教材"
   - 选择小学英语 (PEP) 对应年级

2. **人教社数字教材中心**: https://jczx.pep.com.cn/
   - 注册账号后可下载官方数字资源

3. **国家中小学智慧教育平台**: https://www.zxx.edu.cn/
   - 免费注册后可访问教材资源

### 步骤 2：复制单词数据

从电子课本中找到每个单元的单词表，例如：

```
Unit 1 Hello!
ruler     /ˈruːlə/     尺子
pencil    /ˈpensl/     铅笔
eraser    /ɪˈreɪzə/    橡皮
crayon    /ˈkreɪən/    蜡笔
book      /bʊk/        书
pencil box /ˈpensl bɒks/ 铅笔盒
bag       /bæɡ/        书包
```

### 步骤 3：填入 CSV 模板

打开 `words-template.csv` 文件，按以下格式填入数据：

```csv
grade,semester,unit_no,unit_name,word,phonetic_uk,phonetic_us,meaning_cn,part_of_speech,example_en,example_cn
3,上，1,Unit 1 Hello!,ruler,/ˈruːlə/,/ˈruːlər/,尺子，n.,I have a ruler.,我有一把尺子。
3,上，1,Unit 1 Hello!,pencil,/ˈpensl/,/ˈpensl/,铅笔，n.,This is my pencil.,这是我的铅笔。
```

**重要提示**：
- 使用英文逗号 (,) 作为分隔符，不要使用中文逗号 (,)
- 确保文件保存为 UTF-8 编码
- 年级 (grade) 填写数字：3/4/5/6
- 学期 (semester) 填写：上/下
- 单元号 (unit_no) 填写数字：1-6

### 步骤 4：运行导入脚本

```bash
# 安装依赖（首次运行）
pip install pyodbc pandas openpyxl

# 导入 CSV 文件
python import_words.py your-data.csv

# 或导入 Excel 文件
python import_words.py your-data.xlsx
```

### 步骤 5：验证导入结果

```bash
# 查看题库统计
curl http://localhost:5000/api/seed/stats
```

## CSV 文件格式说明

### 必填字段

| 字段 | 类型 | 说明 | 示例 |
|------|------|------|------|
| grade | 整数 | 年级 | 3, 4, 5, 6 |
| semester | 文本 | 学期 | 上，下 |
| unit_no | 整数 | 单元号 | 1, 2, 3, 4, 5, 6 |
| unit_name | 文本 | 单元名称 | Unit 1 Hello! |
| word | 文本 | 单词 | ruler |
| phonetic_uk | 文本 | 英式音标 | /ˈruːlə/ |
| phonetic_us | 文本 | 美式音标 | /ˈruːlər/ |
| meaning_cn | 文本 | 中文意思 | 尺子 |
| part_of_speech | 文本 | 词性 | n., v., adj., adv. |
| example_en | 文本 | 英文例句 | I have a ruler. |
| example_cn | 文本 | 中文例句 | 我有一把尺子。 |

### 可选字段

- `sort_order`: 排序号（如不填将按顺序自动生成）

## 常见问题

### Q: 导入时提示"单元不存在"
A: 确保数据库 `tb_grade_unit` 表中已有对应的年级单元数据。可以先运行年级单元导入。

### Q: 中文乱码怎么办？
A: 确保 CSV 文件保存为 UTF-8 编码。在 Excel 中另存为时选择"UTF-8 CSV"格式。

### Q: 导入速度慢怎么办？
A: 大批量数据导入时，建议分批进行，例如按年级或学期分批导入。

### Q: 如何删除已导入的数据？
A: 可以使用 SQL 删除：
```sql
-- 删除特定单元的单词
DELETE FROM tb_word WHERE grade_unit_id IN (
    SELECT id FROM tb_grade_unit WHERE grade=3 AND semester='上' AND unit_no=1
);
```

### Q: 音标如何输入？
A: 可以从在线音标编辑器复制，如 https://www.ipachart.com/

## 技术支持

如有问题，请检查：
1. Python 版本是否为 3.8+
2. 是否安装了必要的依赖：`pip install pyodbc pandas openpyxl`
3. SQL Server 服务是否正在运行
4. 数据库 `EnglishLearning` 是否存在
5. CSV 文件格式是否正确（UTF-8 编码，英文逗号分隔）
