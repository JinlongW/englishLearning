## 九、数据验证 SQL

### 查询多释义单词
```sql
SELECT TOP 20 
    word, 
    meaning_cn,
    meaning_trans
FROM tb_word
WHERE JSON_VALUE(meaning_trans, '$[1]') IS NOT NULL;
```

### 查询单元单词分布
```sql
SELECT 
    gu.grade,
    gu.semester,
    gu.unit_no,
    COUNT(w.id) as word_count
FROM tb_grade_unit gu
LEFT JOIN tb_word w ON gu.id = w.grade_unit_id
GROUP BY gu.grade, gu.semester, gu.unit_no
ORDER BY gu.grade, gu.semester, gu.unit_no;
```

### 查询语法点
```sql
SELECT 
    gu.grade,
    gu.semester,
    gu.unit_no,
    g.title,
    g.content_json
FROM tb_grammar g
JOIN tb_grade_unit gu ON g.grade_unit_id = gu.id
ORDER BY gu.grade, gu.semester, gu.unit_no;
```
